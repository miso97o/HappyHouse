# HappyHouse
공공데이터와 카카오 맵 API를 사용한 아파트 실거래가, 주변 상권, 뉴스 정보 확인 사이트
# 프로젝트 설명
## 배경
한 사이트에서 아파트에 대한 정보를 한번에 쉽게 얻을 수 있도록 구성
## 목적
- 회원 가입과 관심 지역 등록을 토대로 한 재검색
- 내가 등록한 관심지역의 주변 상권정보와 환경정보
- 최신 부동산 뉴스
